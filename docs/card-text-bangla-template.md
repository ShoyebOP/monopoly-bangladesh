## Translation Guidelines

1. **Keep Numbers Clear**: Use Arabic numerals (1, 2, 3) or Bangla numerals (১, ২, ৩) consistently
2. **Currency**: Use ৳ symbol or write "Taka"
3. **Tone**: Formal but accessible Bangla
4. **Length**: Keep text concise for card readability
5. **Cultural Adaptation**: Replace US-specific references with Bangladeshi equivalents
