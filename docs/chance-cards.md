## Cultural Adaptations Made

| US Reference | Bangladeshi Equivalent | Rationale |
|--------------|----------------------|-----------|
| Reading Railroad | Kamalapur Railway Station | Main railway station of Dhaka |
| Boardwalk | Bashundhara | Most expensive residential area |
| Stock | Share/Stock | Same concept, widely understood |
| School tax | School tax | Education fees are common |
| Building loan | Building loan | Common in urban Bangladesh |
| Crossword competition | Cricket match competition | Cricket is more popular than crossword in Bangladesh |
| Chairman of the Board | Chairman of the Board | Corporate term, understood in urban areas |

## Effect Type Legend

| Type | Description |
|------|-------------|
| Move | Move token to specified space |

| Pay | Pay money to bank |
| Card | Receive a card item |

## Notes

- All money values use Taka (৳) symbol in Bangla text
- Movement instructions are in simple Bangla
- Cultural references adapted for Bangladeshi context (cricket instead of crossword)
- Location names match the board spaces (Kamalapur, Bashundhara, Dhanmondi, Tejgaon)
